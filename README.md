# Personal Portfolio Website

This is my personal portfolio website created using **HTML, CSS, and JavaScript**.  
It is used to showcase my skills, projects, and contact details as a **Full Stack Java Developer**.

---

## About the Project

This portfolio website helps recruiters and visitors understand:
- Who I am
- What skills I have
- What projects I worked on
- How to contact me

The website is fully responsive and works on mobile, tablet, and desktop devices.

---

## Features

- Responsive design
- Smooth scroll animations
- Typing text animation
- Project section with images and descriptions
- Downloadable resume
- Contact form
- GitHub and LinkedIn links

---

## Technologies Used

- HTML
- CSS
- JavaScript

### Libraries
- AOS (for animations)
- Typed.js (for typing effect)
- Boxicons (for icons)
- Web3Forms (for contact form)

1. Download or clone the repository
2. Open the project folder
3. Open `index.html` in any web browser
